## Product Story

### Section title

Workflow first. Automation underneath it.

### Body

Tenio is not a generic agent or a simple scraper. It is the workflow layer where claim-status work gets done, with automation feeding the workflow by retrieving, interpreting, and proposing candidate outcomes.
